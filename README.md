**Free Resistor Color Code Calculator - Join the Resistance!**

A clean, brutalist-inspired resistor calculator that converts resistance values to color bands and vice versa.

## Features

- **Bidirectional Conversion**: Pick colors to see resistance, or enter a value to see color bands
- **4-band & 5-band Support**: Toggle between standard and precision resistors
- **Real-time Visualization**: See the resistor with colored bands update as you work
- **Standard Value Detection**: Warns when values don't match E12 series
- **Clean Interface**: Brutalist design with maximum usability
- **Zero Dependencies**: Pure HTML, CSS, and JavaScript - no frameworks needed

## Usage

1. **Select colors**: Click on each band to choose a color and see the resistance value
2. **Enter value**: Type a resistance (e.g., "4.7k", "100", "1M") and click Convert to see the color bands
3. **Switch modes**: Toggle between 4-band and 5-band resistor types

## Quick Start

Simply open `index.html` in any modern browser. No build process, no installation required.

```bash
git clone <repository-url>
cd camaradas
open index.html
```
